## 4. Technical SEO Fixes

### 🔴 High Priority

#### Fix 1: Title Tag Length
Current title is 79 characters. Google truncates at ~60.  
**Action:** Shorten to ≤60 characters on all pages (see Section 2).

#### Fix 2: HTTP/2 Protocol
Vercel supports HTTP/2 by default, but confirm it's enabled.  
**Action:** Add this to `vercel.json`:
```json
{
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        { "key": "X-Content-Type-Options", "value": "nosniff" },
        { "key": "X-Frame-Options", "value": "DENY" },
        { "key": "Cache-Control", "value": "public, max-age=31536000, immutable" }
      ]
    }
  ]
}
```

#### Fix 3: Canonical URL Mismatch
Current canonical points to `https://busymulticare.com/` but site is deployed on Vercel.  
**Action:** Either:
- Move domain to `busymulticare.com` (strongly recommended for SEO), OR
- Update all canonical tags to `https://busy-multicare-software-8eqh.vercel.app/`

**Best practice:** Use a custom domain. Add `busymulticare.com` to Vercel project settings.

---

### 🟡 Medium Priority

#### Fix 4: Image Optimization
- Convert all images to **WebP** format
- Add `width` and `height` attributes to all `<img>` tags (prevents layout shift)
- Compress images — use [Squoosh](https://squoosh.app) or `sharp` in build pipeline
- Lazy-load below-the-fold images: `<img loading="lazy" ...>`

#### Fix 5: Core Web Vitals
- **LCP (Largest Contentful Paint):** Preload hero image:
  ```html
  <link rel="preload" as="image" href="/hero-image.webp">
  ```
- **CLS (Cumulative Layout Shift):** Always set image dimensions
- **FID/INP:** Minimize JavaScript blocking time

#### Fix 6: Remove Inline CSS
Move all inline `style=""` attributes to external CSS file. This reduces HTML size and improves caching.

## 11. Performance Fixes

### Mobile & Desktop PageSpeed (Target: 90+ Score)

#### Step-by-step:
1. **Test now:** [pagespeed.web.dev](https://pagespeed.web.dev) — enter your URL
2. **Compress images** → use WebP, set `width`/`height` on all `<img>` tags
3. **Defer JS:** Add `defer` or `async` to non-critical scripts
4. **Preconnect to external domains:**
   ```html
   <link rel="preconnect" href="https://fonts.googleapis.com">
   <link rel="preconnect" href="https://www.google-analytics.com">
   ```
5. **Use Vercel Edge Network** — already fast, ensure static assets cached
6. **Minify CSS/JS** — Vercel does this automatically in production builds

### Inline Styles (SEO + Performance)
Move all `style=""` attributes into a CSS class. Example:
```html
<!-- ❌ Bad -->
<div style="color: red; font-size: 18px;">

<!-- ✅ Good -->
<div class="hero-text">
```
